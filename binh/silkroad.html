Silkroad Origin Mobile (MMO)

Unity developer - GihOt Studio

Project Overview

Giving the players the original and nostalgic feeling of the Silkroad Origin game to the mobile world is what we have been trying to archive.

Using: Unity, C#, NGUI.

Website: https://sromobile.com/home

My Contributions

  • Quest system: Our quest system is quite linear: you complete the required quest to open another. A quest is completed when you pass some conditions, such as having enough item X, killing enough monster Y, going to Z,...
  • Alchemy system: an item upgrading system.
  • Item using recommendation system: quick open/using/sale/... pop up for ease interact with new gaining item.
  • Investigating and fixing several memory leak issues.
