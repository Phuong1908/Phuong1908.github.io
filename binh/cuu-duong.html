Project Overview

Cửu Dương Truyền Kỳ is an immersive MMO set in a rich Chinese wuxia fiction world. Players embark on epic quests, master martial arts, and engage in thrilling battles in a beautifully crafted universe.

My Contributions

  • Pet System

    Developed an attribute upgrade system for pets, enhancing player engagement and character progression.

  • Marriage System

    Implemented an attribute upgrade system for player marriages, adding depth to character relationships and progression.

  • Dueling System

    Created a non-conditional 1v1 PvP system allowing players level 60+ to initiate duels with others on the world map.

  • Bug Fixing

    Identified and resolved various gameplay and performance issues to ensure a smooth gaming experience.

  • Game Content Expansion

    Configured and implemented expanded game content, including new pets, quests, and level progressions.
